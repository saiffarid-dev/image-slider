# 🚀 Image Slider

A modern and responsive **Image Slider** built using **HTML, CSS, JavaScript, and Swiper.js**. It features smooth transitions, navigation arrows, pagination, infinite looping, and a responsive design for all devices.

## ✨ Features

- 🎞️ Smooth image transitions
- 📱 Fully responsive design
- ⬅️ Previous & Next navigation
- 🔵 Pagination dots
- ♾️ Infinite loop

```bash

```

Go to the project folder:

```bash
cd image-slider
```

Open `index.html` in your browser.
